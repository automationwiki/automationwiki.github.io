xCode Mac Setup

  1. Home
  2. xCode Mac Setup
Steps Procedures
1 Before proceeding please install JDK and download eclipse and follow below procedure.
a JDK: Download JDK
b ECLIPSE:Download Eclipse
c Install Xcode from the AppStore and SignIn with Developer Credentials in Accounts Section
d Install Chrome Browser if not Already Installed
e https://drive.google.com/drive/folders/10x3GsGOrDuZ_Z-XlkV_Nxf0iT29dUYdO?usp=sharing
2 First Install Home-brew to manage a package in osX just copy paste the below command:

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"


3 Install command line executer on mac using terminal by using below command:

xcode-select --install


4 Install carthage it is a part of os package command mentioned below:

brew install carthage


5 Install ruby before that just check ruby is installed or not by using ruby -v if not give below command:

\curl -sSL https://get.rvm.io | bash -s stable --ruby


6 Install node using terminal:

brew install node


7 Install Appium doctor:

npm install -g appium-doctor


8 Install appium through npm

npm install -g appium


9 Below command install missing packages for homebrew just copy past one by one:
a brew tap qatools/formulas
b brew install allure-commandline
10 How to set Java home path:

export JAVA_HOME=/Library/Java/Home


11 To install lib mobile devices follow below command:

brew install libimobiledevice


12 After all the steps we need to configure WebDriverAgentRunner in to our xcode. Go to the below path: open Application in mac and right on Appium and follow the path:

/Applications/Appium.app/Contents/Resources/app/node_modules/appium/node_modules/appium-xcuitest-driver/WebDriverAgent/WebDriverAgent.xcodeproj
OR
/Applications/Appium.app/Contents/Resources/app/node_modules/appium/node_modules/appium-webdriveragent/WebDriverAgent.xcodeproj


13 Double click on this WebDriverAgent.xcodeproj then this will open in xcode then follow below procedure:

WebDriverAgent.xcodeproj
