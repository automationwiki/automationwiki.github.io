2 How to Record logs of Android using logcat?

Run the following command "Adb logcat -v threadtime >logcat.txt" and reproduce the issue
3 ADB command to find the list of package name on the device that is connected?

adb shell cmd package list packages
4 How to Install an apk file using Android ADB Commands

adb -s "<deviceIDfromlist>" install "<path-to-apk>"
5 adb command to get the activity of the curent app in focus

adb shell dumpsys window windows | grep -E 'mCurrentFocus'

adb shell dumpsys window | find "mCurrentFocus"
